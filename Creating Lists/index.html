A little history

Tim Berners Lee is credited with creating the world wide web and HTML.
The World Wide Web (W3) initiative links related information throughout the globe. HTML provides one simple format for providing linked information, and all W3 compatible programs are required to be capable of handling HTML. W3 uses an Internet

Terminlogy

HTML
HyperText Markup Language is the language used to author web pages
HTTP
HyperText Transfer Protocol is the network protocol used to request and send documents between computers
